# Daly College Website (Frontend Assignment)

A modern, responsive, and accessible frontend website built using HTML, CSS, and JavaScript with GSAP animations.

## Tech Stack
- HTML5
- CSS3 (Mobile-first)
- JavaScript (ES6)
- GSAP
- ScrollTrigger

## Features
- Fully responsive (mobile-first)
- Dark / Light mode
- Accessible navigation (ARIA, keyboard support)
- Smooth animations
- Optimized carousels

## Setup Instructions

This project is built using plain HTML, CSS, and JavaScript.
No framework or build tool is required.

Method 1: Run Locally (Recommended)
Download or clone the repository:
git clone <repository-url>
